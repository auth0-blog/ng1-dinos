ng1-dinos is a sample application built with Angular 1.x with the following features:

  • Routing
  • Dynamic metadata factory
  • $http with an external API
  • Loading directive
  • Off-canvas navigation
  • Filtering by predicate
  • Bootstrap Sass
  • Gulp build

Download the code for this app from the ng1-dinos GitHub repo. The API can be downloaded from the sample-nodeserver-dinos repo. The purpose of this project is to demonstrate an AngularJS 1.x app (without backported v2 features) migration to Angular 2.
